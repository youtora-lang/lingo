OVERVIEW
What is MJP?
MJP (Makoto Japanese Project) is a vocabulary-building system. Sam shares anything — a single word, a phrase, a full conversation screenshot, something heard on TV, a message from a friend — in Japanese or English. Makoto explains it and helps turn it into vocabulary for chiilingo.JP.
Goal: Turn everyday Japanese input into vocabulary — explained, organized, and ready to practice.
HOW TO USE
2 Modes
💬 GUIDE MODE
Share anything, in any language:
・A single word or phrase you encountered
・A sentence or paragraph in Japanese
・A screenshot of a conversation (describe it in text)
・Something heard on TV, at work, or from a friend
・An English question about Japanese

Makoto explains it in English and picks out words worth remembering.
🍎 LINGO MODE
Type lingo → Export today's key words and phrases as JSON for chiilingo.JP flashcard practice.
GETTING STARTED
How to Start
  • 1
    Copy the prompt
    Press the Copy Prompt button below to copy it to your clipboard.
  • 2
    Create a project on Claude.ai
    Tap + New Project in the left sidebar. Name it MJP or anything you like.
  • 3
    Paste into Project Instructions
    Open Edit or Project Instructions, paste the copied prompt, and save.
  • 4
    Start talking
    Open + New Chat inside the project — share anything in Japanese or English and Makoto will guide you.
