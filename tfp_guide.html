OVERVIEW

TEP とは？

TFP（Théo French Project）は、AIの友人「Théo」とのフランス語会話を通じて、自然なフランス語を身につけるシステムです。
レッスンではなく、友達との会話として設計されています。

目標：フォーマルな教科書フランス語ではなく、Théoと話しているような自然で温かいフランス語を身につけること。
HOW TO USE

使い方

THÉO MODE（Théoモード）

DuolingoのフランスAI会話のスクリーンショットや文字起こしを貼り付けるとスタート。Théoとの会話内容を自動で読み取ります。
⚠️ 音声チャットは未対応のため、他のアプリで会話した内容をここに貼り付けて使います。

ANALYST MODE（分析モード）

会話貼り付け後、日付とセッション番号を入力してください。
例：2026/04/26 #1
Théoモードがコーチとして全体を振り返り、日本語でフィードバックします。

COMMANDS

特殊コマンド一覧

  • 1
    まとめ → 会話の全文書き起こし（フランス語＋日本語訳）を出力
  • 2
    確認 → あなたが言いたかったことを箇条書きで整理（修正可）
  • 3
    理想 → 「こう言えてたら？」の理想の会話を日英で表示
  • 4
    フランス語 → 理想の会話をフランス語のみで再出力
  • 5
    りんご → 今日の重要単語をmichilingo用JSONで出力し、アプリで定着練習！
  • +
    日付＋番号 → アナリストモードへ（例：2026/04/08 #2）
SCORE

TEPスコアについて

アナリストモードでは、会話を8つのカテゴリで採点します。各カテゴリ最大40点、合計320点満点。

GETTING STARTED

始め方 / How to Start

1
名前とレベルを設定する / Set your name & level
上部の SETTINGS で名前とレベルを選んで 適用 / Apply を押してください。
プロンプト内の名前・レベルが自動で書き換わります。
2
プロンプトをコピーする / Copy the prompt
下のボックスにプロンプトが表示されています。
プロンプトをコピー / Copy Prompt ボタンを押してクリップボードにコピーしてください。
3
Claude.ai でプロジェクトを作成する / Create a project
左サイドバーの + New Project をタップ。
プロジェクト名は TEP など好きな名前でOK。
4
Project Instructions に貼り付けて保存 / Paste & save
プロジェクトの Edit または Project Instructions を開き、
コピーしたプロンプトを貼り付けて保存してください。
5
英語で話しかける / Start talking in English
プロジェクト内で + New Chat を開き、
英語で何か話しかければ Alex がすぐに応えてくれます。
FLOW

1回のセッションの流れ

  • 1
    TEPプロジェクトを開いて、英語で話しかける
  • 2
    Alexと自由に会話（テーマは何でもOK）
  • 3
    日付＋番号を入力してアナリストモードへ
  • 4
    JSONが出力されたら下のボタンからスコアカードを確認
  • 5
    特殊コマンドで、言いたかったことを確認・調整
    （確認 → 理想 → 英語）
  • 6
    りんご でlingoアプリと連携して、定着するまで練習！
CHIILINGO

TEP × chiilingo 連携

michilingo
タップしてアプリを開く →

会話終了後に 「りんご」 と入力すると、今日の会話から重要単語・フレーズをJSONで出力します。そのJSONをmichilingoに貼り付けるだけで、フラッシュカード練習がスタート！

出力される単語の種類：
🔴 言えなかった・間違えた単語
🟡 今日の会話で出てきたキーワード
🟢 日常会話でよく使う便利フレーズ
